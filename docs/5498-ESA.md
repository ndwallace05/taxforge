# Form 5498-ESA Implementation

## Overview

This document describes the implementation of IRS Form 5498-ESA in the TaxForge system.

## Components

### Interview View

- **Path**: `/components/5498-ESA/Interview.tsx`
- **Purpose**: Step-by-step data collection interface
- **Usage**: `<5498-ESAInterview onComplete={handleComplete} />`

### Form View

- **Path**: `/components/5498-ESA/Form.tsx`
- **Purpose**: Pixel-perfect replica of official IRS form
- **Usage**: `<5498-ESAForm printMode={true} />`

## Data Flow

1. User enters data in Interview view
2. Redux slice manages state and validation
3. Calculation engine applies Tax-Calculator policies
4. Form view displays final formatted output
5. E-File XML generation for submission

## Testing

Run tests with:

```bash
pnpm run test:form "5498-ESA"
```
